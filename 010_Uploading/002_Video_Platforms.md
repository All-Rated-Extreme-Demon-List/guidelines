## Video Platforms {:supported-video-providers}

The following is a list of websites the AREDL allows you to upload your completion video to. The submission page will not allow you to submit records with an invalid completion link!

- [YouTube](https://youtube.com/)
- [Vimeo](https://vimeo.com/)
- [Twitch](https://twitch.tv/)
- [BiliBili](https://bilibili.com/)
- [Medal](https://medal.tv/)
- [Outplayed](https://outplayed.tv)

### Url Normalization

When submitting a record, the website may normalize your link into the default format for the corresponding platform. In the list below, the “Primary” format is what your URL will be normalized to. If you get an error while submitting a record with a supported provider, check the URL to make sure it is formatted properly, according to the list below.

**YouTube:**

- `https://www.youtube.com/watch?v=<id>` (Primary)
- `https://youtu.be/<id>`
- `https://www.youtube.com/shorts/<id>`
- `https://www.youtube.com/live/<id>`

*The following YouTube hosts are supported:*

- `www.youtube.com`
- `youtube.com`
- `m.youtube.com`

**Vimeo:**

- `https://vimeo.com/<id>` (Primary)
- `https://player.vimeo.com/video/<id>`

*The following Vimeo hosts are supported:*

- `vimeo.com`
- `www.vimeo.com`
- `player.vimeo.com`

**Twitch**

- `https://www.twitch.tv/videos/<id>` (Primary)
- `https://www.twitch.tv/<channel>/video/<id>`
- `https://www.twitch.tv/<channel>/v/<id>`
- `https://player.twitch.tv/?video=v<id>`

*The following Twitch hosts are supported:*

- `twitch.tv`
- `www.twitch.tv`
- `player.twitch.tv`

**BiliBili**

- `https://www.bilibili.com/video/<id>` (Primary)

*The following BiliBili hosts are supported:*

- `bilibili.com`
- `www.bilibili.com`
- `m.bilibili.com`

**Medal**

- `https://medal.tv/clips/<id>` (Primary)
- `https://medal.tv/<language>/clips/<id>`
- `https://medal.tv/games/<game>/clips/<id>`
- `https://medal.tv/<language>/games/<game>/clips/<id>`

*The following Medal hosts are supported:*

- `medal.tv`
- `www.medal.tv`

**Outplayed**

- `https://outplayed.tv/media/<id>` (Primary)
- `https://outplayed.tv/<game>/<id>`*
    - ** Note that `<game>` can be any string of text.*

*The following Outplayed hosts are supported:*

- `outplayed.tv`
- `www.outplayed.tv`


If you would like to suggest additional platforms for video submissions, please reach out a team member with your recommendations.
